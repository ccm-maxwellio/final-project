# final-project

# Overview

The project analyzes data on automobile recalls, determining which parts and manufacturers are susceptible to recalls, as well as the trends of the number of recalls over the past few years. The goal of this project is to provide manufacturers with insight into the reliability of their products, as well as which parts of their vehicles they should focus R&D on.

# Purpose

The purpose of this project is to evaluate vehicle reliability and safety by identifying historical patterns in automotive recalls. By utilizing the data rather than general consumer perception, this project aims to uncover which auto manufacturers face the most frequent safety issues, which specific vehicle components are most prone to failure, and whether the overall volume of industry recalls is improving or worsening over time.
